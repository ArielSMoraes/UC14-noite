## Seletores CSS

*O que é um seletor css?*

Básicamente é uma regra que se aplica a um ou varios elementos html dispostos na tela, estas regras tem potuações e hierarquias


* Elementos 1 ponto

* Pseudo classe 10 pontos

* Classe 10 pontos

* Id 100 pontos

* !importante 1000 pontos
